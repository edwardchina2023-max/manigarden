## 更新方式

网站内容源位于《行路不难》项目工作区。重新构建源网站后，在本仓库运行：

```bash
node scripts/export-static.mjs /absolute/path/to/website/dist
```

导出脚本会生成适配 `/manigarden/` 子路径的纯静态页面，并保留 `.nojekyll`。

## 内容边界

公开站点只保存已分级的公开资料，不存证件、完整录音、联系方式、精确位置、未授权影像或完整账本。AI未生成任何冒充现场的纪实影像。
